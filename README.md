# LikeMyke Finance Dashboard 🚀

A premium, modern Personal Finance Landing Page built with **Flutter**. This project demonstrates a clean and professional UI design for financial management, featuring real-time balance tracking, transaction history, and intuitive categorization.

---

## ✨ Features

- **📊 Comprehensive Financial Overview**: Interactive balance card displaying total balance, monthly income, and expenses.
- **💸 Transaction Management**: A detailed list of recent transactions with visual indicators for income/expense types.
- **🏷️ Smart Categorization**: Horizontal scrollable category chips (Food, Tech, Health, Travel, Leisure) for quick filtering.
- **🔍 Search Functionality**: Integrated search bar for easy transaction lookup.
- **📱 Responsive & Modern UI**: Built with Material 3, custom gradients, and a sleek Teal/Slate color palette.
- **🎨 Premium Typography**: Leverages Google Fonts (Outfit for headings and Inter for body text) for superior readability.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/) (v3.8.1+)
- **Language**: [Dart](https://dart.dev/)
- **Design System**: Material 3
- **Typography**: [Google Fonts](https://pub.dev/packages/google_fonts) (Outfit, Inter)
- **Icons**: Cupertino Icons & Material Symbols

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- An IDE (VS Code, Android Studio, or IntelliJ).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/landing-flutter.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd landing-flutter
   ```

3. **Install dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the application**:
   ```bash
   flutter run
   ```

## 🎨 UI Design Details

The application uses a **Premium Design System** defined in `app_theme.dart`:
- **Primary Colors**: Teal 700 (#0F766E) & Slate 900 (#0F172A).
- **Background**: Soft Slate (#F8FAFC).
- **Interactive Elements**: Custom chips with glassmorphism-inspired borders and subtle shadows.
